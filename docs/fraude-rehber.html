Genel Bakış

FRAUDE nedir?

FRAUDE, Borsa İstanbul (BIST) hisselerini izleyen yatırımcı ve analistler için bir masaüstü finans terminalidir. Fiyat ve teknik göstergeleri, bilanço oranlarını, KAP bildirimlerini, haber akışını ve halka arz takvimini tek bir çalışma alanında birleştirir; üstüne kendi AI anahtarınızla çalışan bir araştırma katmanı ekler.

Bu rehber iki şeyi anlatır: (1) uygulamanın tüm modüllerini ne işe yaradıklarıyla birlikte, (2) AI özelliklerini açmak için API anahtarınızı sağlayıcıya göre adım adım nasıl alıp uygulamaya gireceğinizi. Sonda uygulamanın web'e taşınma mimarisi ve yayınlama hattı özetlenir.

AI Kurulumu

API anahtarını sağlayıcına göre ekle

AI Araştırma, İzleme Radarı yorumları ve ajan analizleri senin kendi API anahtarınla çalışır — anahtarı sen alır, uygulamaya girersin, kullanım senin hesabına işler. Aşağıdan sağlayıcını seç; nereye gireceğini, ne alacağını ve FRAUDE'ye nasıl gireceğini adım adım göster.

OpenAI

Ücretli · kullanım bazlıKredi kartı gerekir

GPT modelleri. Genel amaçlı analiz için güçlü varsayılan. Anahtar sk- ile başlar ve yalnızca bir kez gösterilir.

  1. platform.openai.com adresine git, hesap aç veya giriş yap.
  2. Sağ üstteki hesap menüsünden API keys sayfasını aç (platform.openai.com/api-keys).
  3. Create new secret key → anahtara bir ad ver → oluştur.
  4. Çıkan sk-… anahtarını hemen kopyala (sayfayı kapatınca tekrar görünmez).
  5. Billing → Payment methods altından kart ekle ve bir miktar bakiye tanımla (yoksa istekler reddedilir).
Base URL
https://api.openai.com/v1
Modeller
gpt-4ogpt-4o-minio3-minio1o1-mini

DeepSeek

Ücretli · çok uygunÖn ödemeli bakiye

DeepSeek-V3 (sohbet) ve DeepSeek-R1 (akıl yürütme). Düşük maliyetle güçlü analiz isteyenler için. OpenAI-uyumlu API.

  1. platform.deepseek.com adresine git ve kayıt ol.
  2. Panelde API keys bölümünü aç.
  3. Create new API key → oluştur → sk-… anahtarını kopyala.
  4. Top up / Billing altından hesabına bakiye yükle.

https://api.deepseek.com/v1

Google Gemini

Ücretsiz katman varGoogle hesabı yeter

Gemini modelleri. Ücretsiz katmanla denemeye başlamak için en kolay yol. FRAUDE, Google'ın OpenAI-uyumlu ucunu kullanır.

  1. aistudio.google.com (Google AI Studio) adresine Google hesabınla gir.
  2. Sol menüden veya üstten Get API key'e tıkla.
  3. Create API key → (istenirse bir proje seç) → anahtarı kopyala.
  4. Daha yüksek limit istersen faturalandırmayı (Cloud Billing) etkinleştir; başlangıç için ücretsiz katman yeterli.

https://generativelanguage.googleapis.com/v1beta/openai/

Qwen · Alibaba Cloud

ÜcretliDeneme kotası

Alibaba'nın Qwen modelleri, DashScope üzerinden. Uzun bağlam ve çok dilli görevlerde güçlü. compatible-mode = OpenAI-uyumlu uç.

  2. Konsolda dashscope.console.aliyun.com → API-KEY yönetimini aç.
  3. Create API Key → sk-… anahtarını kopyala.
  4. Uluslararası hesaplarda uç dashscope-intl… olabilir; sağlayıcının verdiği base URL'i esas al.

Custom · OpenAI-uyumlu

Base URL sen girersinClaude · Llama · Mistral

Custom seçeneği, OpenAI-uyumlu herhangi bir uca bağlanmanı sağlar: base URL ve model kimliğini sen belirlersin. Anthropic Claude, Meta Llama veya Mistral gibi modelleri tek anahtarla sunan bir ağ geçidi (ör. OpenRouter, Together AI) ya da kendi proxy'in.

  1. Bir sağlayıcı seç — ör. openrouter.ai (tek anahtarla Claude/Llama/Mistral) veya together.ai.
  2. Hesap aç → Keys / API Keys → yeni anahtar oluştur → kopyala.
  3. Sağlayıcının Base URL'ini not al (ör. OpenRouter: https://openrouter.ai/api/v1).
  4. Kullanmak istediğin tam model kimliğini sağlayıcının model listesinden al (aşağıdaki örnekler FRAUDE'de hazır gelir).
Base URL
senin sağlayıcın (Custom alanına yapıştır)
Örnek modeller
claude-3-7-sonnet-latestclaude-3-5-sonnet-latestmeta-llama/Llama-3-70b-chat-hfmistralai/Mistral-7B-Instruct-v0.2

Not: Model kimliği sağlayıcıya göre değişir (ör. OpenRouter'da anthropic/claude-3.7-sonnet). Doğru dizeyi mutlaka sağlayıcının belgelerinden kopyala.

Sonra: FRAUDE içinde

Settings›AI Providers›Add key

Provider'ı seç (OpenAI / DeepSeek / Google / Qwen / Custom), bir Label yaz, kopyaladığın API key'i yapıştır, Default model'i seç. Provider'ı seçince Base URL otomatik dolar (Custom'da elle girersin). Kaydettikten sonra Test ile doğrula ve istersen varsayılan yap. Birden fazla anahtar ekleyip ajanlarına farklı anahtar atayabilirsin.

Güvenlik: Anahtar kaydedildikten sonra arayüze düz metin olarak geri dönmez; yalnızca maskeli gösterilir (sk-t••••). Web sürümünde anahtarlar sunucuda şifreli saklanacak.

Başla

Hızlı başlangıç

Masaüstünde denemek

  1. Depoyu klonla ve npm install ile bağımlılıkları kur.
  2. npm run tauri dev ile uygulamayı aç (Rust + React birlikte derlenir).
  3. İlk açılışta piyasa verisi otomatik senkronlanır; Dashboard dolmaya başlar.
  4. AI için Settings › AI Providers'a git ve yukarıdaki adımlarla anahtarını ekle.
  5. İzleme Radarı'na birkaç hisse ekle ve enabled yap; arka plan taraması başlasın.

Web'e taşımaya başlamak

  1. Projeyi git init ile versiyon kontrolüne al (şu an git deposu değil).
  2. Vercel · Supabase · Fly.io hesaplarını aç.
  3. Faz 0: backend'i Tauri'den ayırıp axum /v1/rpc kabuğunu kur.
  4. Frontend'i VITE_FRAUDE_API_URL ile lokal backend'e bağlayıp uçtan uca doğrula.
